### Run the Application
#### Local
+ Client
```
npm install
npm start
```
+ [Prisma](https://www.prismagraphql.com/docs/quickstart/)
```

npm start
```

### Deploying
#### [Zeit](https://zeit.co/)
+ Client
```
now && now alias
```

+ [Prisma](https://www.prismagraphql.com/docs/quickstart/)
```
npm install -g prisma
prisma deploy
now && now alias
```
